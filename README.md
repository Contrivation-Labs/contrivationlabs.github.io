# Contrivation Labs — Company Website

Live site: [contrivation-labs.github.io/contrivationlabs.github.io](https://contrivation-labs.github.io/contrivationlabs.github.io/)

Bangalore-based Agritech start-up focused on Harvest and Post-Harvest Solutions for the Indian Agriculture sector.

## Structure

```
contrivationlabs.github.io/
├── index.html       # Page structure and content
├── styles.css       # All styles and layout
└── media/           # Logos, product images, award badges
```

## Sections

| Section | Description |
|---------|-------------|
| Hero | Tagline and call-to-action |
| About | Company overview, stats, and vision |
| Awards & Grants | BIRAC, Startup Karnataka, Pusa Krishi recognitions |
| Products | CalyXtract® — automated chili calyx separation system |
| Contact | Location, email, phone, social links, and inquiry form |

## Adding content

**New product** — add a `.product-card` block in the Products section of `index.html`.

**New award** — add a `.award-card` block in the Awards section of `index.html`.

**Style changes** — edit `styles.css` only.

**Contact details** — search for `TODO` in `index.html` to find the email and phone placeholders.

**Contact form** — sign up at [formspree.io](https://formspree.io), create a form, and replace `YOUR_FORM_ID` in the form's `action` attribute.
